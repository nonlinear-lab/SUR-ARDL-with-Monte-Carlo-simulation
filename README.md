# SUR-ARDL-with-Monte-Carlo-simulation
this repository contains R-codes for SUR-ARDL analysis including Monte Carlo simulation of critical values of I(0) and I(1)
1 R-codes for data generating process, common movement rho=0.8, 1. white noise, 2 AR=0.5, 3. AR=0.8, 4. AR=0.95, 5. randam walk
2 R-codes for ARDL analysis with ardl function
3 R-codes for ARDL analysis with manual calculation
4 R-codes for ARDL analysis with manual calculation of OLS analysis
5 R-codes for SUR analysis with systemfit function
6 R-codes for SUR analysis with manual calculation
7 R-codes for SUR-ARDL analysis with systemfit function
8 R-codes for SUR-ARDL analysis with "sur_ardl_bounds" function
9 R-codes for Monte Carlo simulation for SUR-ARDL analysis for I(0)
10 R-codes for Monte Carlo simulation for SUR-ARDL analysis for I(1)
